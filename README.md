# flext
---
## [A weight lifting helper app](https://flext-app.herokuapp.com/)

### Built with
- [React](https://reactjs.org/)
- [Redux](https://redux.js.org/)
- [D3](https://d3js.org/)
- [Firebase](http://firebase.google.com/)
- [Webpack](https://webpack.js.org/)

### To start development server run
```
npm run dev-server
```
